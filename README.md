# [ Node Shop](https://github.com/itcast-wh/node-shop)

## 技术栈

    node.js + express + middleware + morgan + mysql

## 下载源码
    git clone https://github.com/itcast-wh/node-shop.git

## 开始使用

#### 安装依赖

    npm install

#### 启动项目

    npm start

## 线上接口文档地址(域名: litc.pro 暂未备案结束)
